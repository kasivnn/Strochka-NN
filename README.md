# Строчка-НН

Статический промо-сайт для авторской мастерской по разработке и пошиву модельной и современной одежды.

## Файлы

- `index.html` — структура страницы;
- `styles.css` — оформление и адаптивность;
- `script.js` — мобильное меню и анимация появления блоков.

## Как открыть

Просто откройте файл `index.html` в браузере.
